Note

This is a simple color mixing tool using the Kubelka-Munk theory; it is not perfect and only serves as a simple approximation. It operates under CIE D65, D50, and A illuminant. The pigment data wavelength interval is 10 nm.

Warning: Any non-textual color representation on the webpage is accurate. This is due to the nature of displaying colors in sRGB or HEX formats on a webpage, which are approximations of the intended color.

For example, the pigment labeled as cobalt blue is displayed as rgb(63, 68, 245), even though the intended value is rgb(44, 65, 247), as the text shows.

If you use the color picker to sample colors directly from the webpage, it will capture the already inaccurately rendered color correctly. In other words, it generates the correct textual representation of the wrong graphic representation on the website (rgb(63, 68, 245) for cobalt blue). Then representing this correct textual representation in graphic format in the color picker display square introduces further distortion due to another round of sRGB-to-web display conversion.

The coordinates of the xy chromaticity diagram are designed to correctly represent the XYZ tristimulus values calculated from Kubelka-Munk result (i.e., reflectance). Whenever any clipping occurs during these calculation, a console log will warn and show the value before clipping.
